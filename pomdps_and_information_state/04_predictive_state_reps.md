# Predictive State Representations

## The Reorientation

The belief (01) summarizes history by a posterior over a **latent**
state — it presumes the generative model. PSRs (Littman–Sutton–Singh
2002) summarize history by predictions about the **observable future**:
no latent ontology, only forecasable quantities.

A **test** is a sequence `t = a_1 o_1 a_2 o_2 ... a_k o_k`; its
prediction from history `h` is

```math
p(t\mid h)=\Pr\{\text{observations of }t\text{ occur}\mid h,\ \text{actions of }t\text{ taken}\}.
```

## The System-Dynamics Matrix And The Rank Theorem

Arrange all histories against all tests:

```math
\mathcal{D}\in\mathbb{R}^{\text{histories}\times\text{tests}},
\qquad
\mathcal{D}[h,t]=p(t\mid h).
```

**Theorem.** If `rank(D) = k` (finite), there exist `k` **core tests**
`Q = {q_1..q_k}` whose prediction vector
`b(h) = (p(q_i|h))_i in R^k` is a sufficient statistic: for every test
`t` there is a weight vector `m_t` with

```math
p(t\mid h)=\big\langle m_t,\ b(h)\big\rangle\quad\forall h,
```

and `b` updates linearly-normalized after each `(a, o)`:

```math
b(h\,ao)_i
=
\frac{p(ao\,q_i\mid h)}{p(ao\mid h)}
=
\frac{\langle m_{ao q_i},\,b(h)\rangle}{\langle m_{ao},\,b(h)\rangle} .
```

*Proof.* Rank `k` means `k` linearly independent columns — the core
tests; every column (test) is a linear combination of them, giving
`m_t`. The update: `ao q_i` and `ao` are themselves tests, hence linear
in `b(h)`; Bayes' rule for "prediction of `q_i` after experiencing
`ao`" is exactly the displayed ratio. ∎

**Corollary (PSR vs belief).** A POMDP with `|S| = n` has
`rank(D) <= n` (the belief is a linear bottleneck:
`p(t|h) = <u_t, b_belief(h)>` — condition on the hidden state), so the
PSR dimension never exceeds the minimal latent-state dimension. Strict
inequality is possible: **PSRs can be strictly more compact than any
belief representation.**

## The Compression Example

The float/reset flavor: an environment whose observable process is a
deterministic-ish cycle observable through `k` tests, generated by a
hidden chain that must track additional distinctions (e.g. a hidden
counter modulo `m` that the observation channel projects down). The
belief must maintain the full posterior over `n = ` (cycle × counter)
latent states — distinctions that **never become observable** — while
`rank(D)` collapses to the observable cycle's dimension: the PSR stores
only what the future can ever reveal. The general principle:

```text
belief dimension  = what the MODELER distinguished
PSR dimension     = what the FUTURE distinguishes
                    (<= the former, sometimes <<)
```

— sufficiency measured against prediction, not against ontology.

## Spectral Learning

Because everything is linear behind a normalization, PSRs admit
**method-of-moments** learning with no EM, no local optima. Estimate
from data the matrices

```math
\hat P_{\mathcal{H},\mathcal{T}}
\quad(\text{history-test co-occurrence probabilities}),
\qquad
\hat P_{\mathcal{H},ao,\mathcal{T}}
\quad(\text{the same, with }ao\text{ interposed}),
```

SVD the first: `P_{H,T} ~ U S V^T` of rank `k` reveals the core
subspace; the per-`(a,o)` update operators are recovered by the
**transfer identity**

```math
B_{ao}
=
U^\top\hat P_{\mathcal{H},ao,\mathcal{T}}\,V S^{-1}
\qquad\text{(shapes chosen so } b(hao)\propto B_{ao}\,b(h)\text{)},
```

with consistency and finite-sample bounds inherited from matrix
concentration + SVD perturbation (Davis–Kahan) — the
`concentration_toolkit/` pipeline applied to Hankel-like matrices (this
is the same spectral technology as HMM learning, Hsu–Kakade–Zhang).
**Load-bearing:** the SVD step needs a *rank gap* (singular value
`sigma_k` bounded away from 0 — poorly observable distinctions =
vanishing singular values = the statistics blow up as `1/sigma_k`:
hardness (03) resurfacing as conditioning, not undecidability).

## Position And Interfaces

```text
to linear_mdps_and_completeness/04:  low-rank D is a bilinear-class-type
    structural assumption; PSR-learnability results (recent: poly-sample
    online PSRs) sit exactly in that program's frame;
to 05:  the AIS view subsumes both beliefs and PSRs as instances of
    "a statistic whose predictions close" — PSRs are the minimal LINEAR
    such statistic;
to practice:  transformers trained on next-token prediction are learning
    predictive statistics of history by construction — the PSR
    philosophy (represent by forecasts, not by posited latents) is
    arguably the one this era's architectures adopted, without the
    linear algebra.
```

## What Remains Open

Nonlinear PSRs with guarantees (the linear rank theory is tight but
narrow); action-conditional exploration for PSR learning (the moment
matrices need data under diverse action sequences — an experiment-design
problem); and the exact relationship between transformer in-context
state tracking and finite-rank predictive statistics — currently
empirical folklore awaiting its theorem.
